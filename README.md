# 📊 Financial Analysis Dashboard

## 📌 Overview

The **Financial Analysis Dashboard** is an interactive Business Intelligence dashboard developed using **Microsoft Power BI**. It provides a comprehensive overview of financial performance by analyzing sales, profit, gross sales, discounts, units sold, product performance, country-wise sales, and monthly business trends using Microsoft's Financial Sample dataset.

This project demonstrates data visualization, KPI reporting, dashboard design, and business intelligence concepts through an interactive dashboard that transforms raw financial data into meaningful business insights.

---

# 🎯 Why I Built This Project

As an aspiring **Data Analyst** and **Data Scientist**, I wanted to strengthen my Business Intelligence and data visualization skills by building an interactive Power BI dashboard.

The primary goal of this project was to transform raw financial data into meaningful business insights that can support decision-making. Through this dashboard, I explored how organizations monitor Key Performance Indicators (KPIs), compare product and regional performance, and identify business trends using visual analytics.

This project also allowed me to gain hands-on experience with Power BI, Power Query, DAX, dashboard design principles, and data storytelling while building a professional portfolio for internships and future career opportunities.

# 📊 Dataset

**Dataset:** Microsoft Power BI Financial Sample Dataset

The Financial Sample dataset is a sample business dataset provided by Microsoft Power BI. It contains sales transactions across different countries, products, business segments, and time periods, making it suitable for learning Business Intelligence, dashboard development, and data visualization techniques.
